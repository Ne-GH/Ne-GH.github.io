栈(LIFO)
=
---

首先要在C++源文件中包含``<stack>``头文件
定义方式如下:
``stack<int>Q``表示定义了一个栈,栈名为S,队列中的元素均为int型

栈的函数有:
|函数|作用|
|:-:|:-:|
|``pop()``|出栈|
|``push()``|入栈|
|``top()``|访问栈顶元素|
|``empty()``|栈空则返回真,否则为假|
|``size()``|返回栈中元素的个数|
程序样例:
```C++
#include<iostream>
#include<queue>
using namespace std;
int main(){

    stack<int>S;
    S.push(1);
    S.push(2);
    S.push(3);
    cout<<"栈长度: "<<S.size()<<endl;
    cout<<"S为空吗: "<<S.empty()<<endl;
    cout<<"栈顶元素为: "<<S.top()<<endl;
    S.pop();
    S.pop();
    S.pop();
    cout<<"栈长度: "<<S.size();
    return 0;
}


```
